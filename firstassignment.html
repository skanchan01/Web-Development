ABOUT-US
--------------

Smart Field History Tracker – By CtrlF Systems

Smart Field History Tracker is a field auditing tool built on top of Salesforce by CtrlF Systems that enforces governance by tracking History data within Salesforce. This tool will help you track your field level changes for any object in a few configuration steps. We help organizations to audit their sensitive data easily and report it at any time without using salesforce storage space.
